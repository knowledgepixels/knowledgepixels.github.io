# Knowledge Pixels Website

Static website for [Knowledge Pixels](https://knowledgepixels.com), built with plain HTML, CSS, and modern JavaScript, and hosted on GitHub Pages.

News items are published as [nanopublications](https://nanopub.net/) and rendered from published nanopubs using [nanopub-js](https://github.com/nanopublication/nanopub-js) for querying, fetching, and processing.
