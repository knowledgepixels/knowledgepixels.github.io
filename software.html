Our Software

This is our software to support you with nanopublication technology.

Nanodash

Nanodash is our user interface and API provider for dealing with nanopublications. It provides user-friendly templates for nanopublication creation, user-specific feeds, search, and journal connection to formally publish nanopublications with academic articles.

GitHub

nanopub-java

Nanopub-java is a Java library to deal with nanopublications. This is currently the most complete implementation of nanopublications, covering hash-based identifier generation, digital signatures, nanopublication indexes, and more.

GitHub

Nanopub Query

Nanopub Query is our second-generation service for querying the published nanopublications with SPARQL. It divides the nanopublications per user profile and per nanopublication type into different triple stores, so queries to these triple stores are not slowed down by unrelated nanopublications.

GitHub

Nanopub Registry

Nanopub Registry is our second-generation service for publishing and retrieving nanopublications. It is still under development, and will replace the first-generation nanopub servers once it is fully implemented.
